What is MSIX?

MSIX is a modern Windows app package format, a successor of APPX and a "spiritual successor" to some other - now legacy - formats, including but not limited to MSI, App-V and ClickOnce.

Apps deployed as MSIX packages benefit from several out-of-the-box features and properties, including:

  • Fast and reliable installation, clean uninstallation,
  • Content delivery optimizations, ensuring faster downloads with minimum impact and network bandwidth usage,
  • Access to various modern Windows APIS which are not available for classically deployed win32 apps,
  • Extensibility and shimming thanks to Package Support Framework,
  • Add-ons and modification packages,
  • Easy switch to MSIX app attach,
  • Distribution and monetization in Microsoft Store.

And best of all, the big part of the ecosystem is open-sourced and the core support for MSIX deployment is built-in in any computer running still supported version of Windows 10. Although the format is relatively young, there already exist several package authoring tools, both from Microsoft and ISVs, and the community is constantly growing.

Going to MSIX format (either repackaging or app modernization) preserves the functionality of existing app packages and for a straightforward conversion it requires no access to the original sources.

MSIX HERO Features

This app helps with management, testing, trobuleshooting and simply understanding your MSIX portfolio:

  • Displaying, finding and managing installed packages,
  • Installing packages, bundles and manifests (in developer mode).
  • Provisioning / de-provisioning of packages (installation for all users),
  • Visualizing dependencies and Package Support Framework configuration,
  • Displaying users who installed the package,
  • Creating and extracting certificates,
  • Packing / unpacking MSIX packages,
  • Signing and updating package identities,
  • Troubleshooting and tools: starting commands in package context, revealing actual folders, loading registry etc.,
  • Detection of authoring tools (Visual Studio, MSIX Packaging Tool, Advanced Installer, InstallShield, RayPack etc.
  • Creating modification packages and .appinstaller files.
  • NEW IN 0.3.0.0 Creating VHD expanded MSIX for app attach scenarios (preview).

This list is not comprehensive, and the app itself is frequently updated with new additions and bug fixes. Report your ideas and issues via e-mail ten.orehxism@olleh.A detailed history of previous changes is available here.
