MSIX Hero

The ultimate MSIX toolkit for IT Pros

Features

  • Viewing, installing and uninstalling MSIX/APPX/bundles
  • Provisioning / de-preovisioning of packages
  • Visualizing dependencies and Package Support Framework configuration
  • Creating and extracting certificates
  • Packing / unpacking MSIX packages
  • Signing and updating package identities
  • Troubleshooting and tools: starting commands in package context, opening folders, loading registry etc.
  • Detection of authoring tools (Visual Studio, MSIX Packaging Tool, Advanced Installer, RayPack, InstallShield etc.
  • Creating modification packages and .appinstaller files
